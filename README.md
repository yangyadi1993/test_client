# Design and implement a simple dashboard demo
The project structure was initialized by `create-react-app`.

## Quick Start

Clone project and install dependencies:
```bash
$ git clone https://github.com/yangyadi1993/test_client.git
$ cd test_client
$ npm install
```

Start the app:
```bash
$ npm start -s
```

Build the app:
```bash
$ npm run build
```
